About the Game | 2D Pixel Adventure

Rainfall is an atmospheric 2D pixel adventure game set in a fantasy universe. Master 4-way combat movement, explore beautiful biome environments, and uncover forgotten lore as you fight the Corrupted Kings in order to save your Twin.

2D Pixel Art & Game Development

Building a 2D action fantasy world requires meticulous precision.

Crafted Pixel Art

Every environment, character frame, and blade of grass in Rainfall is crafted using dedicated 2D pixel art techniques, focusing on atmospheric tilesets.

Top-Down Mechanics

Features smooth 4-way movement blueprints and custom physics to bring the dense, dangerous terrain to life.

Pixel Design

Bridging classic pixelated aesthetics with modern mechanics taking ideas from raw concept sketches to detailed sprite sheets animated frame by frame.
